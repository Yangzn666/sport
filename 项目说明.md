# 健身学习与数据追踪系统 🏋️‍♂️

## 系统架构

本系统分为两个独立的网站：

### 1️⃣ 公共知识库 📚
- **位置**：`fitness-knowledge-base/`
- **端口**：`http://localhost:5173`
- **用途**：健身科学知识文献聚合平台
- **特点**：基于PubMed文献，公开访问
- **功能**：
  - 📖 健身科学知识浏览
  - 🔍 文献搜索与分类
  -  5大知识分类、73篇文献

### 2️⃣ 个人数据追踪器 🔒
- **位置**：`personal-fitness-tracker/`
- **端口**：`http://localhost:5182`
- **用途**：您的私人训练数据分析
- **特点**：完全本地运行，数据隐私保护
- **功能**：
  -  数据可视化（跑步、力量、体成分）
  - 💡 智能分析报告
  - 🎯 个性化训练建议

## 数据管理

### 数据来源
所有数据文件位于：
```
fitness-knowledge-base/data/your_data/
├── your_running_data.csv              # 跑步记录（279条）
├── your_strength_data.csv             # 力量训练（1605组）
└── your_body_composition_data.csv     # 体成分数据（6次测量）
```

### 同步数据到个人追踪器
双击运行：`同步数据.bat`

或手动执行：
```bash
xcopy "fitness-knowledge-base\data\your_data\*" "personal-fitness-tracker\public\your_data\" /E /I /Y
```

### 数据备份
```bash
xcopy "fitness-knowledge-base\data\your_data\*" "D:\Backup\FitnessData" /E /I /Y
```

## 网站功能详解

### 公共知识库 📚

#### 知识分类
1. **运动生物力学** - 动作力学原理
2. **运动生理学** - 身体适应机制
3. **运动营养学** - 饮食与补充
4. **训练方法学** - 训练计划设计
5. **运动心理学** - 心理调控技巧

#### 特色功能
- 按类别浏览文献
- 关键词搜索
- 文献详情查看（作者、PMID、摘要）
- 响应式设计，手机友好

### 个人数据追踪器 🔒

#### 仪表盘
- 快速统计数据卡片
- 跑步/力量/体成分概览
- 功能入口导航

#### 数据可视化 (`/analytics`)
- **跑步分析**：
  - 月度跑量统计
  - 配速趋势（7天/30天移动平均）
  - 训练类型分布饼图
  - 心率区间分析

- **力量训练**：
  - 部位训练分布（推/拉/腿/核心）
  - 月度训练量趋势
  - 动作容量排行
  - 训练频率统计

- **体成分追踪**：
  - 体重变化曲线
  - 体脂率趋势
  - 骨骼肌变化
  - 基础代谢追踪

#### 智能分析 (`/smart-analysis`)
- **总览**：关键指标汇总
- **跑步分析**：最佳成绩、训练分布、改进建议
- **力量分析**：部位分布、优势与短板
- **体成分分析**：变化趋势、评估建议
- **综合建议**：
  - 训练方案（含考研期间优化版）
  - 营养指导（蛋白质/碳水/脂肪）
  - 恢复管理（睡眠/主动恢复/压力管理）

## 数据安全

### 隐私保护措施
✅ 所有数据存储在本地电脑  
✅ 网站运行在localhost，无需联网  
✅ 不收集、不上传、不共享任何数据  
✅ 可随时备份或删除所有数据  
✅ 与公共知识库完全隔离  

### 数据备份建议
1. **定期备份**到云盘或外部硬盘
2. **版本控制**（可选）：使用Git管理CSV文件
3. **多地存储**：至少保留2份备份

## 常见问题

### Q1: 两个网站可以同时运行吗？
**A**: 可以！它们运行在不同端口，互不干扰：
- 公共知识库：`http://localhost:5173`
- 个人追踪器：`http://localhost:5182`

### Q2: 如何更新训练数据？
**A**: 
1. 编辑 `fitness-knowledge-base/data/your_data/` 中的CSV文件
2. 运行 `同步数据.bat` 同步到个人追踪器
3. 刷新浏览器即可看到更新

### Q3: 数据安全吗？会被上传吗？
**A**: 完全安全！所有数据仅在您的本地电脑上，不会上传到任何服务器。

### Q4: 可以在手机上访问吗？
**A**: 可以，但需要手机和电脑在同一网络：
1. 在电脑上运行 `npm run dev -- --host`
2. 手机访问显示的局域网IP地址

### Q5: 如何停止网站？
**A**: 在终端按 `Ctrl+C`，或直接关闭终端窗口。
