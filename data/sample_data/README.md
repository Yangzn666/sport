# 运动数据导入模板

## 跑步数据模板 (running_data.csv)

```csv
日期,距离,时间,配速,心率,类型
2026-01-01,5.0,25,5.0,150,轻松跑
2026-01-03,8.0,40,5.0,155,节奏跑
2026-01-05,5.0,23,4.6,160,间歇跑
2026-01-08,10.0,52,5.2,145,长距离
2026-01-10,6.0,28,4.7,158,轻松跑
```

### 字段说明
- **日期**: 格式 YYYY-MM-DD
- **距离**: 公里 (km)
- **时间**: 分钟 (min)
- **配速**: 分钟/公里 (min/km)
- **心率**: 平均心率 (bpm)
- **类型**: 轻松跑、节奏跑、间歇跑、长距离、比赛等

---

## 健身数据模板 (strength_data.csv)

```csv
日期,动作,重量,组数,次数,容量,部位
2026-01-01,深蹲,80,4,8,2560,腿部
2026-01-01,卧推,60,4,8,1920,胸部
2026-01-01,杠铃划船,50,4,10,2000,背部
2026-01-03,硬拉,100,3,6,1800,背部
2026-01-03,哑铃肩推,20,3,10,600,肩部
2026-01-05,深蹲,85,4,8,2720,腿部
2026-01-05,卧推,62.5,4,8,2000,胸部
```

### 字段说明
- **日期**: 格式 YYYY-MM-DD
- **动作**: 动作名称（深蹲、卧推、硬拉等）
- **重量**: 公斤 (kg)
- **组数**: 训练组数
- **次数**: 每组次数
- **容量**: 重量 × 组数 × 次数 (kg)
- **部位**: 腿部、胸部、背部、肩部、手臂等

## 数据来源建议

### 跑步数据
- **Strava**: 导出 GPX/TCX 文件后转换为 CSV
- **Garmin Connect**: 导出活动数据
- **Keep**: 手动记录训练日志
- **运动手表**: 华为、佳明、Apple Watch 等

### 健身数据
- **Strong App**: 专业的力量训练记录应用
- **Hevy**: 健身追踪应用
- **Excel/Notion**: 手动记录训练日志
- **传统训练日记**: 数字化您的纸质记录

## 使用步骤

1. **准备数据**: 按照模板格式整理您的运动数据
2. **导出 CSV**: 确保文件编码为 UTF-8
3. **访问分析页面**: 点击首页的"运动数据可视化分析"
4. **导入数据**: 点击"导入数据"按钮，选择 CSV 文件
5. **查看报告**: 自动生成可视化图表和分析建议

---

## 注意事项

- CSV 文件第一行必须是表头（字段名）
- 日期格式统一为 YYYY-MM-DD
- 数值字段不要包含单位（如 "5km" 应写为 "5"）
- 中文字段名和英文字段名都支持
- 缺失数据可以留空，系统会自动过滤
