# 2026世界杯中国队“四皇一神”人物设定

## 👑 核心四人组

### 1. 叶泽辉 (Ye Zehui) - “优雅的躲球艺术家”
*   **位置**：左后卫 / 中场
*   **球员模板**：保罗·马尔蒂尼 (防守优雅) + 佩德里 (技术细腻/脑回路清奇)
*   **性格特点**：
    *   **社恐型天才**：极度害怕身体对抗，看到壮汉冲过来会下意识闭眼。
    *   **脑洞大开**：过人方式永远在对手预料之外（比如突然停下来系鞋带晃倒对方）。
    *   **点球软脚虾**：大赛关键时刻罚点球必腿抖，心理抗压能力较弱。
*   **技术风格**：防守稳如老狗，进攻骚气冲天。擅长用意想不到的“脚滑”或“假摔式”变向过掉世界级边锋。

### 2. 黄佳炀 (Huang Jiayang) - “时间静止的领袖”
*   **位置**：中场 / 右边锋
*   **球员模板**：托尼·克罗斯 (上帝视角/精准长传) + C罗 (领袖气质/大心脏)
*   **性格特点**：
    *   **球队大脑**：中国队史第二人，拥有统御全场的霸气。
    *   **节奏大师**：速度偏慢，但善于利用节奏变化让对手怀疑人生。
    *   **关键先生**：越是绝境越冷静，拥有“大心脏”属性。
*   **技术风格**：传控俱佳，大局观强。他的传球像装了导航系统，总能找到最致命的空档。

### 3. 陈家乐 (Chen Jiale) - “锦鲤中锋”
*   **位置**：中锋 / 中后卫
*   **球员模板**：埃尔林·哈兰德 (身体怪兽) + 哈里·马奎尔 (转身慢/运气好)
*   **性格特点**：
    *   **天赋异禀**：身高1.9米，身体天赋一流，但过人能力几乎为零。
    *   **玄学射手**：射门全靠蒙，但每次都能蒙进世界波。
    *   **红颜祸水**：沉迷女色，容易受到场边美女啦啦队的影响而分心。
*   **技术风格**：站桩输出，运气爆棚。经常在背对球门的情况下，用脚后跟磕出神仙球。

### 4. 杨臻宁 (Yang Zhening) - “咆哮的手抛球狂魔”
*   **位置**：门将
*   **球员模板**：彼得·舒梅切尔 (恐怖咆哮/出击) + 吉安路易吉·布冯 (神级反应)
*   **性格特点**：
    *   **反差萌**：身高不高，但反应神速，拥有恐怖的爆发力。
    *   **点球专家**：点球大战的心理战大师，压力对他来说就是兴奋剂。
    *   **手榴弹投手**：大力手抛球能直接砸晕对方前锋，甚至直接助攻得分。
*   **技术风格**：善于出击1对1化解单刀，飞扑动作极具观赏性，但脚下出球能力较弱，经常把回传球踢出界。

## 🧠 传奇教练

### 方毅 (Fang Yi) - “上古神人的执念”
*   **身份**：中国队主教练
*   **球员模板**：贝利 (球王光环) + 马拉多纳 (不羁灵魂)
*   **背景故事**：
    *   **中国队历史第一人**：年轻时天赋冠绝全球，但因横遭不幸（被外星人绑架去踢星际联赛）没能带领国足走向辉煌，壮志难酬。
    *   **复仇之火**：退役后把所有精力都放在了这支“怪才”队伍上，誓要弥补当年的遗憾。
*   **执教风格**：
    *   **哲学训话**：每次赛前训话都像在讲神话故事，被网友做成“方毅语录”表情包。
    *   **战术鬼才**：擅长根据对手的弱点制定极其离谱但有效的战术（比如让陈家乐去盯防对方的美女经纪人）。
