## 🐛 常见问题排查

### 搜索无结果？
- [ ] 检查query是否转为小写
- [ ] 确认knowledgeContents已加载
- [ ] 测试简单关键词（如"的"）

### 目录不显示？
- [ ] 检查Markdown标题格式（#后需空格）
- [ ] 确认正则表达式正确
- [ ] 查看console有无报错

### 进度未保存？
- [ ] 检查localStorage容量（上限5-10MB）
- [ ] 确认浏览器未禁用Cookie
- [ ] 查看Storage面板数据

### 代码不高亮？
- [ ] 确认highlight.js文件存在
- [ ] 检查`<code>`标签class是否正确
- [ ] 调用hljs.highlightAll()时机
