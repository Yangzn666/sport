## 🔍 按问题查找

### 训练相关

**Q: 我应该多久练一次？**
→ 《力量训练科学》- "全身训练 vs 分化训练"
- 初学者：每周3-4次全身训练
- 中级：每周4-6次分化训练
- 您的情况：考研期间，每周3次全身训练✅

**Q: 每组做多少次最好？**
→ 《力量训练科学》- "特异性原则"
- 力量：1-5次 @ 85-95% 1RM
- 肥大：6-12次 @ 70-85% 1RM
- 耐力：15+次 @ <70% 1RM

**Q: 组间休息多久？**
→ 《运动生理学基础》- "能量系统"
- ATP-CP系统：3-5分钟
- 糖酵解系统：1-2分钟
- 有氧系统：30-60秒

**Q: 什么时候应该增加重量？**
→ 《力量训练科学》- "渐进超负荷"
- 当你能以标准动作完成目标次数上限时
- 增幅：2.5-5% / 周
- 示例：80kg × 10次 → 82.5kg × 8次

### 跑步相关

**Q: 我应该用什么配速训练？**
→ 《有氧训练与耐力科学》- "训练强度分区"
- 轻松跑：Z2（70-80% HRmax），可轻松交谈
- 阈值跑：Z3（80-90% HRmax），勉强说短句
- 间歇跑：Z4-Z5（90-100% HRmax），无法说话

**Q: 如何提高半马成绩？**
→ 《有氧训练与耐力科学》- "耐力三要素"
1. 提升VO₂max：4×4分钟间歇
2. 提升乳酸阈值：20-30分钟阈值跑
3. 改善跑步经济性：力量训练+plyometrics

**Q: 我应该脚跟跑还是前掌跑？**
→ 《有氧训练与耐力科学》- "着地方式争议"
- 科学共识：没有一种适合所有人
- 如无伤痛，保持自然着地方式
- 重点改善步频（目标180 spm）

**Q: 每周应该跑多少公里？**
→ 《有氧训练与耐力科学》- "10%规则"
- 新手：从20-30km/周开始
- 每周增幅不超过10%
- 每3-4周安排减负周

### 恢复相关

**Q: 我需要休息多久？**
→ 《运动生理学基础》- "超量恢复原理"
- 轻度训练：24小时恢复
- 中度训练：48小时恢复
- 重度训练：72小时恢复
- 每4-6周安排1周减负

**Q: 睡眠对训练有多重要？**
→ 《运动生理学基础》- "肌肉适应机制"
- 生长激素主要在深度睡眠分泌
- 建议每晚7-9小时
- 睡眠不足会降低睾酮，升高皮质醇

**Q: 训练后多久吃东西？**
→ 待补充（营养篇章正在编写中）
- 传统观点：30分钟"合成窗口"
- 最新研究：24小时内总摄入更重要
- 建议：训练后1-2小时内进食

### 受伤预防

**Q: 膝盖疼怎么办？**
→ 《有氧训练与耐力科学》- "常见跑步受伤"
- 可能是髌股疼痛综合征（Runner's Knee）
- 原因：臀中肌无力、髌骨轨迹异常
- 预防：加强臀部力量、改善步态

**Q: 小腿酸痛正常吗？**
→ 《运动生理学基础》- "肌肉损伤"
- 轻微DOMS（延迟性肌肉酸痛）正常
- 通常在训练后24-72小时达到峰值
- 如疼痛剧烈或持续>1周，需就医

**Q: 如何判断是否过度训练？**
→ 《运动生理学基础》- "ACWR负荷监控"
- ACWR > 1.5：过度训练风险高
- 静息心率持续升高
- 睡眠质量下降
- 情绪烦躁、动力降低
