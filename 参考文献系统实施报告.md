## ⚠️ 当前限制

### 需要手动补充的信息
1. ❌ **作者信息**：大部分文献缺少完整作者列表
2. ❌ **发表年份**：需要从DOI或标题中提取
3. ❌ **期刊名称**：需要补充完整期刊信息
4. ❌ **证据等级**：需要人工评估并标注
5. ❌ **关键发现**：需要阅读摘要后总结

### 技术限制
1. ⚠️ **PubMed链接缺失**：当前主要依赖DOI
2. ⚠️ **经典文献不足**：缺少1950s-2000s的奠基性研究
3. ⚠️ **验证未进行**：部分DOI可能失效或错误

#### 文献卡片渲染
```javascript
function createReferenceCard(ref) {
    return `
        <div class="reference-card">
            <h3>
                <a href="https://doi.org/${ref.doi}" target="_blank">
                    ${escapeHtml(ref.title)}
                </a>
            </h3>
            <div class="ref-authators">${ref.authors.join(', ')}</div>
            <div class="ref-journal">${ref.journal} (${ref.year})</div>
            <span class="evidence-level ${getEvidenceLevelClass(ref.evidence_level)}">
                ${ref.evidence_level}
            </span>
            <div class="ref-links">
                <a href="https://doi.org/${ref.doi}" target="_blank">📄 DOI</a>
                <a href="https://pubmed.ncbi.nlm.nih.gov/${ref.pubmed_id}/" target="_blank">🔬 PubMed</a>
            </div>
        </div>
    `;
}
```

#### 搜索功能
```javascript
document.getElementById('searchInput').addEventListener('input', (e) => {
    const keyword = e.target.value.toLowerCase().trim();
    const filtered = allReferences.filter(ref => 
        ref.title.toLowerCase().includes(keyword) ||
        (ref.authors && ref.authors.some(a => a.toLowerCase().includes(keyword)))
    );
    renderReferences(filtered);
});
```
