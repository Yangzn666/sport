## 🎯 项目概览

### 项目名称
**健身与跑步科学知识库** (Fitness & Running Science Knowledge Base)

### 项目定位
一个**基于证据的健身科学知识库**，整合运动生理学、力量训练、有氧耐力、营养恢复等领域的权威学术文献，提供可追溯、可验证的科学训练指导。

### 核心价值
- 📚 **权威性**：所有内容由81+篇正式学术文献支持
- 🔗 **可追溯**：每个论点都可点击跳转到原始文献
- 🎨 **美观性**：学术风格设计，思源宋体排版
- 📱 **易用性**：响应式设计，支持桌面和移动端
- 🔄 **可扩展**：模块化架构，易于维护和扩展

### 目标用户
- 💪 健身爱好者（初级到高级）
- 🏃 跑步运动员
- 🎓 体育专业学生
- 👨‍🏫 健身教练
- 🔬 运动科学研究者

```javascript
// 搜索功能
document.getElementById('searchInput').addEventListener('input', (e) => {
    const keyword = e.target.value.toLowerCase();
    const filtered = allReferences.filter(ref => 
        ref.title.toLowerCase().includes(keyword) ||
        ref.authors.some(a => a.toLowerCase().includes(keyword))
    );
    renderReferences(filtered);
});

// 文献卡片渲染
function createReferenceCard(ref) {
    return `
        <div class="reference-card">
            <h3><a href="https://doi.org/${ref.doi}" target="_blank">
                ${ref.title}
            </a></h3>
            <div class="ref-authators">${ref.authors.join(', ')}</div>
            <span class="evidence-level level-1a">${ref.evidence_level}</span>
        </div>
    `;
}
```
