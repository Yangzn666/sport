# 《荒诞与救赎》网页版使用说明

## 📖 小说名称优化

### 新名称：**《荒诞与救赎：四皇的世界杯传奇》**

#### 命名理念：
1. **"荒诞"** - 体现余华风格的荒诞美学
   - 叶泽辉的闭眼转圈防守
   - 黄佳炀的时间静止

2. **"救赎"** - 体现陀思妥耶夫斯基的救赎主题
   - 叶泽辉从恐惧到勇气的救赎
   - 黄佳炀从懒惰到智慧的救赎
   - 陈家乐从迷茫到自由的救赎
   - 杨臻宁从愤怒到确认的救赎
   - 方毅从逃避到传承的救赎

3. **"四皇"** - 突出四个核心人物
4. **"世界杯传奇"** - 保留故事核心

- ✅ **响应式设计**：适配手机和平板

## 📊 技术细节

### HTML结构
```html
<header>         <!-- 标题和副标题 -->
<nav>            <!-- 章节导航 -->
<main>           <!-- 主要内容区 -->
  <section id="intro">      <!-- 故事简介 -->
  <section id="characters"> <!-- 人物介绍 -->
  <section id="ch1">        <!-- 第一章 -->
  ...
  <section id="ch12">       <!-- 第十二章 -->
</main>
<footer>         <!-- 页脚引用 -->
```

### CSS特色
- **渐变背景**：`linear-gradient(135deg, #1a1a2e, #16213e, #0f3460)`
- **毛玻璃效果**：`backdrop-filter: blur(10px)`
- **金色阴影**：`box-shadow: 0 8px 32px rgba(255, 215, 0, 0.2)`
- **平滑过渡**：`transition: all 0.3s ease`
- **动画效果**：`@keyframes fadeIn`

### JavaScript功能
- **粒子生成**：动态创建50个浮动粒子
- **章节切换**：显示/隐藏对应章节
- **进度条更新**：监听scroll事件
- **返回顶部**：平滑滚动到顶部
