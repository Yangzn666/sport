# 四所大学及临安企业真实信息汇总

## 一、浙江警察大学（叶泽辉）

### 学校基本信息
- **正式名称**：浙江警察学院（Zhejiang Police College）
- **性质**：浙江省唯一培养公安专门人才的本科高校
- **校区**：滨江校区 + 临安校区
- **校训**：忠诚、严谨、团结、献身
- **在校生**：4000余人
- **教职工**：489人（专任教师252人）

### 特色专业
- **国家级一流本科专业建设点**（4个）：
  - 涉外警务
  - 网络安全与执法
  - 治安学
  - 交通管理工程

- **省级一流本科专业建设点**（4个）：
  - 侦查学
  - 刑事科学技术
  - 经济犯罪侦查
  - 警务指挥与战术

### 体育设施
- 多功能室内射击靶场等29个实训教学场所
- 警务实战教学训练基地
- 模拟犯罪现场
- 车辆驾驶训练场
- 射击训练馆

### 足球相关
- 实行警务化管理，体能训练严格
- 学生需具备优秀身体素质
- 校内有完善的运动场地

### 可用于小说的细节
1. **警务化管理**：每天早操、队列训练、内务整理
2. **体能要求**：3000米跑、引体向上、仰卧起坐等考核
3. **纪律严明**：迟到一分钟都要受罚
4. **职业导向**：毕业后直接进入公安系统
5. **临安校区**：位于锦北街道集贤村，占地500余亩，总投资约6亿元

## 二、南通大学（黄佳炀）

- **性质**：江苏省人民政府和交通运输部共建的省属综合性大学
- **历史**：始建于1912年，源自张謇创办的私立南通医学专门学校和南通纺织专门学校
- **校区**：啬园校区、启秀校区、启东校区（3个校区）
- **占地面积**：3300余亩
- **建筑面积**：120余万平方米
- **学科**：79个本科招生专业，涵盖10个学科门类
- **ESI排名**：13个学科进入全球前1%（截至2026年3月）

### 体育设施（极其完善）
- **体育馆**：3座
- **羽毛球馆**：1-2座
- **击剑馆**：1座（高水平运动队）
- **室内场馆**：乒乓球房、健美操房、健身房、武术房
- **400米标准田径场**：4片
- **300米田径场**：1片
- **足球场**：6-8片
- **篮球场**：32-40余片
- **网球场**：4-6片
- **排球场**：8-20余片

### 足球队情况
- **高水平运动队**：田径、击剑、乒乓球（三个项目）
- **击剑队成绩**：
  - 国际比赛：3银、4铜
  - 国家级比赛：22金、19银、28铜
  - 省级比赛：13金、15银、28铜
- **田径队成绩**：
  - 国家级比赛：1金4银1铜
  - 省级比赛：12金14银15铜
- **荣誉**：2018年入选"全国体育竞赛百强高校"

### 科研平台
- 国际奥林匹克研究中心1个
- 省部共建体育实验与体测中心等省部级平台3个
- 运动风险评估实验室
- 运动解剖实验室
- 运动生理生化实验室
- 体育保健实验室
- 体质健康测评实验室
- 身体成分分析实验室

### 可用于小说的细节
1. **啬园校区**：主校区，位于南通市崇川区啬园路9号
2. **体育科学学院**：江苏省培养体育师资的主要基地之一
3. **学术氛围**：图书馆周末座无虚席，学长学姐在楼梯口背书
4. **张謇精神**："祈通中西，力求精进"的校训
5. **足球场地充足**：6-8片足球场，训练条件优越
6. **黄佳炀的选择**：放弃名校光环，选择南通大学的体育资源

## 三、浙江农林大学（陈家乐）

### 学校基本信息
- **英文名称**：Zhejiang A&F University（简称ZAFU）
- **性质**：浙江省唯一的省属本科农林类高校
- **共建**：浙江省人民政府 + 国家林业和草原局
- **历史**：创建于1958年，时称天目林学院；1966年更名为浙江林学院；2010年更名浙江农林大学
- **校区**：东湖校区、衣锦校区、诸暨校区（3个校区）
- **占地面积**：3200-3300余亩
- **在校生**：32200余人
- **位置**：杭州市临安区（杭州西郊国家森林城市）

### 临安校区详情
- **衣锦校区**（老校区）：
  - 位置：临安市锦城街道衣锦街252号
  - 面积：330-468亩
  - 特点：地处老城区，设有3个普通学院和干部管理学院
  
- **东湖校区**（新校区/本部）：
  - 位置：临安区武肃街666号
  - 面积：2000-2500亩
  - 特点：现代化生态校园，"两园（校园、植物园）合一"

### 足球文化（非常浓厚）
- **校队成绩**：
  - 男足：浙江省大学生联赛甲组第三名、乙组亚军、省第十三届大运会第六名
  - 女足：浙江省第十六届大运会冠军（连续第三年、累计第五次全省冠军，2020-2022三连冠）
  
- **足球设施**：
  - 四个球场：东湖校区A操、B操（标准十一人制）
  - 衣锦校区：人造草标准十一人制足球场 + 标准五人制场地
  - 每年在东湖B操和衣锦球场举办校园联赛

- **足球协会**（ZAFU FA）：
  - 成立于2010年
  - 部门：赛事部、裁判部、宣传部、办公室、后勤部
  - 培养裁判员：30名现役，50+储备
  - 活动：春季联赛、校友足球邀请赛、手游实况足球比赛

- **校友足球邀请赛**：
  - 2023年11月18日举办第一届（建校65周年献礼）
  - 70余名校友参赛
  - 4支队伍：校教工明星队、三七二一队、校友明星队、球朋旧友队
  - 赛前进行三轮运动点球增加趣味性

### 学科特色
- **国家级一流专业建设点**：14个
- **国家级特色专业**：4个
- **浙江省登峰学科**：1个
- **浙江省一流学科**：9个
- **国家重点实验室**：浙江省属高校首个

### 校园环境
- "国家生态文明教育基地"
- "浙江省生态文化研究中心"
- "两园合一"现代化生态校园被誉为"浙江省高校校园建设的一张亮丽名片"

### 可用于小说的细节
1. **临安本地高校**：就在临安区，离家近
2. **足球氛围浓厚**：女足三连冠，男足实力强劲
3. **生态校园**：环境优美，适合思考人生
4. **农林特色**：以农林学科为特色，涵盖九大学科门类
5. **校友网络**：70余名校友参加足球邀请赛，体现凝聚力
6. **陈家乐的选择**：留在临安，远离喧嚣，寻找真正的自己

## 四、浙江中医药大学（杨臻宁）

### 学校基本信息
- **英文名称**：Zhejiang Chinese Medical University（简称ZCMU）
- **性质**：浙江省重点建设高校、高水平大学建设高校
- **共建**：浙江省人民政府 + 国家中医药管理局 + 教育部
- **历史**：始创于1953年6月（浙江省中医进修学校）；1959年成立浙江中医学院；2006年更名浙江中医药大学
- **校区**：滨文校区、富春校区（2个校区）
- **占地面积**：1500余亩
- **在校生**：23000-26000余人（硕博士研究生5200-6000余人）
- **附属医院**：
  - 直属：附属第一医院（浙江省中医院）、附属第二医院（浙江省新华医院）、附属第三医院（浙江省中山医院）
  - 同质化管理：杭州市中医院、温州市中医院、宁波市中医院、省立同德医院

### 体育特长
- **高水平运动队招生**：
  - 武术套路（0-18人）
  - 女子足球（0-10人）
  
- **体育部**：
  - 1996年成立体育工作部，2005年更名为体育部
  - 负责：体育教学与科研、运动队训练与竞赛、群众体育活动、军事理论教学与军训
  - 下设：部办公室、体育教研室（田径球类、民传形体类2个课程组）、群体竞赛中心、学生体质健康与传统养身

### 足球相关
- **女子足球高水平运动队**：
  - 测试内容：身体素质（5*25米折返跑）、基本技术（传准、曲线运球射门）、实战能力
  - 守门员测试：立定三级跳远、掷远与踢远
  - 录取分档：A档（单独招生）、B档（高考65%线）、C档（二本线）

- **男子足球**：虽未列入高水平运动队，但校内足球氛围良好

### 学术地位
- **博士学位授权点**：中医学、中药学、中西医结合、护理学（4个一级学科）
- **博士后科研流动站**：3个（中医学、中药学、中西医结合）
- **硕士学位授权点**：9个一级学科
- **国家级人才**：中国工程院院士、国医大师等

### 可用于小说的细节
1. **中医背景**：学习中医学，研究人体奥秘，探索生命本质
2. **滨文校区**：位于杭州市滨江区滨文路548号，与六和塔隔江相望
3. **体育与中医结合**：传统养身、体质健康监测
4. **杨臻宁的选择**：用中医智慧理解足球，用科学方法训练门将技术
5. **沉默的专注**：中医讲究"静"，与杨臻宁的性格契合
6. **附属医院资源**：可以接触运动医学、康复训练等专业领域

## 六、小说应用场景建议

### 1. 叶泽辉（浙江警察大学）

- **成长**：在张謇"祈通中西"精神下，找到足球与哲学的平衡

### 3. 陈家乐（浙江农林大学）
