# 字体优化说明 🎨

## 使用的字体

### 1️ **主字体 - Noto Sans SC（思源黑体）**
- **来源**：Google Fonts
- **特点**：现代、清晰、易读
- **字重**：300（细）、400（常规）、500（中等）、700（粗）、900（特粗）
- **适用**：正文、标题、数字

### 2️⃣ **备选字体栈**
```css
font-family: 'Noto Sans SC', 
             -apple-system,          /* macOS/iOS */
             BlinkMacSystemFont,     /* macOS */
             'Segoe UI',             /* Windows */
             'PingFang SC',          /* macOS 中文 */
             'Hiragino Sans GB',     /* macOS 旧版中文 */
             'Microsoft YaHei',      /* Windows 中文 */
             sans-serif;             /* 通用回退 */
```

### 3️⃣ **其他字体**
- **Noto Serif SC（思源宋体）**：可用于特殊标题（暂未使用）
- **ZCOOL XiaoWei（站酷小薇）**：艺术字体（暂未使用）

## 字体渲染优化

### ✅ **已应用的优化**

1. **抗锯齿渲染**
   ```css
   -webkit-font-smoothing: antialiased;
   -moz-osx-font-smoothing: grayscale;
   ```
   - 让字体更清晰锐利
   - 减少模糊感

2. **文本渲染优化**
   ```css
   text-rendering: optimizeLegibility;
   ```
   - 优化可读性
   - 更好的字间距和连字

3. **标题样式**
   ```css
   h1, h2, h3 {
     font-weight: 700;
     letter-spacing: -0.02em;  /* 稍微紧凑 */
     line-height: 1.2;          /* 紧凑行高 */
   }
   ```

4. **数字等宽**
   ```css
   .tabular-nums {
     font-variant-numeric: tabular-nums;
     font-feature-settings: "tnum";
   }
   ```
   - 数字对齐更整齐
   - 适合数据展示

5. **滚动条美化**
   - 圆角设计
   - 柔和颜色
   - 悬停效果

## 加载性能

### ⚡ **字体加载策略**
1. **Google Fonts CDN** - 自动优化加载
2. **字体预加载** - 浏览器缓存
3. **字体回退** - 网络慢时使用系统字体
4. **多字重** - 只加载需要的字重

### 📦 **字体文件大小**
- Noto Sans SC：约 200KB（压缩后）
- 首次加载：稍慢（~500ms）
- 后续访问：缓存，秒开

## 自定义建议

### 如果想更换字体：

1. **更圆润的字体**
   ```css
   @import url('https://fonts.googleapis.com/css2?family=Nunito:wght@400;600;700&display=swap');
   font-family: 'Nunito', sans-serif;
   ```

2. **更现代的字体**
   ```css
   @import url('https://fonts.googleapis.com/css2?family=Inter:wght@400;500;700&display=swap');
   font-family: 'Inter', sans-serif;
   ```

3. **更优雅的宋体**
   ```css
   font-family: 'Noto Serif SC', serif;
   ```
