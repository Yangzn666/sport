## 📊 顶级科研网站核心特征分析

### 1️⃣ PubMed（美国国家医学图书馆）

**核心优势：**
- ✅ **极简界面设计** - 白色主调，无广告干扰，视觉聚焦度高
- ✅ **智能搜索系统** - 实时提示MeSH主题词、拼写纠正、搜索建议
- ✅ **多维筛选功能** - 按文献类型、物种、年龄组、语言等即时过滤
- ✅ **个性化服务** - My NCBI账户、检索历史保存、文献收藏夹
- ✅ **响应式设计** - 移动端完美适配
- ✅ **引用导出功能** - 一键生成Citation，支持EndNote/RIS格式

**用户体验亮点：**
```
搜索框 → 智能提示 → 结果筛选 → 单篇详情 → 引用/分享/全文链接
   ↓         ↓          ↓           ↓            ↓
 输入中   MeSH推荐   时间轴过滤   PMID+DOI    一键复制
```

**可借鉴功能：**
1. **智能搜索提示** - 输入时实时推荐相关术语
2. **高级筛选面板** - 多维度快速过滤结果
3. **个人收藏系统** - 保存感兴趣的文档和段落
4. **引用格式生成** - APA/MLA/GB/T 7714等格式

```javascript
class BookmarkManager {
    constructor() {
        this.bookmarks = JSON.parse(localStorage.getItem('bookmarks') || '{}');
    }
    
    // 收藏整个文档
    addDocument(docName) {
        this.bookmarks.documents = this.bookmarks.documents || [];
        if (!this.bookmarks.documents.includes(docName)) {
            this.bookmarks.documents.push(docName);
            this.bookmarks.timestamps = this.bookmarks.timestamps || {};
            this.bookmarks.timestamps[docName] = Date.now();
            this.save();
        }
    }
    
    // 收藏特定段落
    addParagraph(docName, text, note) {
        this.bookmarks.paragraphs = this.bookmarks.paragraphs || [];
        this.bookmarks.paragraphs.push({
            id: generateUUID(),
            doc: docName,
            text: text.substring(0, 100) + '...',
            note: note,
            timestamp: Date.now()
        });
        this.save();
    }
    
    save() {
        localStorage.setItem('bookmarks', JSON.stringify(this.bookmarks));
    }
    
    // 导出为JSON
    exportBookmarks() {
        const blob = new Blob([JSON.stringify(this.bookmarks, null, 2)], 
            { type: 'application/json' });
        const url = URL.createObjectURL(blob);
        const a = document.createElement('a');
        a.href = url;
        a.download = `bookmarks_${new Date().toISOString().split('T')[0]}.json`;
        a.click();
    }
}
```

                markdown += `---\n\n`;
            });
        });
        
        downloadFile(markdown, 'notes.md', 'text/markdown');
    }

// 基于关键词相似度的简单推荐算法
function getRelatedArticles(currentDoc) {
    const currentTerms = extractKeywords(knowledgeContents[currentDoc]);
    const recommendations = [];
    
    Object.keys(knowledgeContents).forEach(docName => {
        if (docName === currentDoc) return;
        
        const docTerms = extractKeywords(knowledgeContents[docName]);
        const similarity = calculateSimilarity(currentTerms, docTerms);
        
        if (similarity > 0.3) {  // 相似度阈值
            recommendations.push({
                doc: docName,
                similarity: similarity,
                sharedTerms: getSharedTerms(currentTerms, docTerms)
            });
        }
    });
    
    return recommendations
        .sort((a, b) => b.similarity - a.similarity)
        .slice(0, 5);  // 返回最相关的5篇
}

class UserProfile {
    constructor() {
        this.profile = JSON.parse(localStorage.getItem('userProfile') || '{}');
    }
    
    // 阅读历史
    addToHistory(docName, scrollPosition) {
        this.profile.history = this.profile.history || [];
        this.profile.history.unshift({
            doc: docName,
            position: scrollPosition,
            timestamp: Date.now(),
            duration: 0  // 需要计时器追踪阅读时长
        });
        
        // 只保留最近50条
        if (this.profile.history.length > 50) {
            this.profile.history.pop();
        }
        
        this.save();
    }
    
    // 阅读统计
    getReadingStats() {
        const history = this.profile.history || [];
        const docCounts = {};
        
        history.forEach(item => {
            docCounts[item.doc] = (docCounts[item.doc] || 0) + 1;
        });
        
        return {
            totalReads: history.length,
            favoriteDocs: Object.entries(docCounts)
                .sort((a, b) => b[1] - a[1])
                .slice(0, 5),
            lastRead: history[0]?.doc || '无'
        };
    }
    
    // 导出统计数据
    exportStats() {
        const stats = this.getReadingStats();
        const csv = convertToCSV(stats);
        downloadFile(csv, 'reading_stats.csv', 'text/csv');
    }
}

### 3. 智能问答机器人 🤖

**功能描述：**
基于知识库内容的本地AI问答（无需后端）。

```javascript
class KnowledgeQA {
    constructor() {
        this.index = buildSearchIndex(knowledgeContents);
    }
    
    // 简单的问题匹配
    answerQuestion(question) {
        // 提取问题关键词
        const keywords = extractKeywords(question);
        
        // 在知识库中搜索
        const relevantDocs = searchByKeywords(keywords);
        
        // 提取相关段落
        const answers = relevantDocs.map(doc => {
            return {
                doc: doc.name,
                snippet: findBestSnippet(doc.content, keywords),
                confidence: calculateConfidence(keywords, doc.content)
            };
        });
        
        // 返回最佳答案
        return answers.sort((a, b) => b.confidence - a.confidence)[0];
    }
}

// 使用示例
const qa = new KnowledgeQA();
const answer = qa.answerQuestion('如何提高最大摄氧量？');
// 返回：{ doc: '有氧训练与耐力科学', snippet: '...', confidence: 0.85 }
```

## 🛠️ 技术选型建议

### 前端框架（可选升级）
目前使用纯HTML/CSS/JS，如需更复杂功能可考虑：

| 框架 | 优势 | 劣势 | 适用场景 |
|-----|------|-----|---------|
| Vue.js | 轻量、易学 | 需要构建步骤 | 中等复杂度应用 |
| React | 生态丰富 | 学习曲线陡 | 大型应用 |
| Svelte | 无虚拟DOM | 生态较小 | 追求性能 |
| **保持现状** | **零依赖** | **功能受限** | **当前最佳选择** |

**建议：** 继续保持纯静态方案，通过Web Components逐步增强功能。

### 数据存储方案

| 方案 | 容量 | 持久性 | 复杂度 | 推荐度 |
|-----|------|-------|-------|-------|
| localStorage | 5-10MB | 永久 | 简单 | ⭐⭐⭐⭐⭐ |
| IndexedDB | 无限制 | 永久 | 中等 | ⭐⭐⭐⭐ |
| Service Worker Cache | 无限制 | 可控 | 复杂 | ⭐⭐⭐ |

**建议：** 
- 短期：继续使用localStorage
- 中期：迁移到IndexedDB（支持更大容量）
- 长期：考虑Service Worker离线缓存
